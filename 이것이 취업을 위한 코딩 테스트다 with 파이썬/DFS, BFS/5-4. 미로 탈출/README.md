# 5-4 미로 탈출

---

## 📖 문제 정보
- **챕터**: DFS/BFS
- **난이도**: ★★☆
- **풀이 시간**: 30분
- **시간 제한**: 1초
- **메모리 제한**: 128MB

---

## 📝 문제 요약

- N X M 크기의 직사각형 형태의 미로에 갇혀 있다
- 동빈이의 위치는 (1, 1)이며 출구는 (N, M)이다
- 괴물이 있는 부분은 0, 괴물이 없는 부분은 1이다
- 한 번에 한 칸씩 이동 가능하며 셀 때 시작 칸과 마지막 칸을 포함해서 계산
- 탈출하기 위해 움직여야하는 최소 칸의 개수를 구하라
